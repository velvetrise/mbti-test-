## 🧠 MBTI гэж юу вэ?

**Myers-Briggs Type Indicator (MBTI)** нь хүний зан чанарыг 4 хэмжээст дагуу ангилдаг сэтгэл зүйн загвар:

| Хэмжээст | Утга |
|----------|------|
| **E / I** | Гадаад эрчим хүч vs Дотоод эрчим хүч |
| **S / N** | Бодит мэдрэхүй vs Хийсвэр интуиц |
| **T / F** | Логик сэтгэлгээ vs Мэдрэмжийн шийдвэр |
| **J / P** | Төлөвлөгөөт амьдрал vs Уян хатан амьдрал |
